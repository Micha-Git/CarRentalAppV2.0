# Use Case "List Cars in Fleet"

```
Title: List Cars in Fleet

Primary Actors: Fleet manager
Secondary Actors: None

Preconditions: None 
    
Postconditions: None

Flow:
1. Fleet manager lists all cars in fleet.
2. System presents all cars in the fleet and their basic car information for the given parameters.

Alternative flows:
1a. The operator has not added any cars to the fleet.
    1. The system displays a message indicating that no cars are currently in the fleet.
    2. The system provides an option or prompt for the operator to add cars to the fleet.

Information Requirements: None
```
