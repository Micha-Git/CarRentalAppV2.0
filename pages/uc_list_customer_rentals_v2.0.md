# Use Case "List Customer Rentals"

```
Title: List Customer Rentals

Primary Actors: Customer
Secondary Actors: None

Preconditions: None

Postconditions: None

Flow:
1. Customer lists his rentals.
2. System presents the customer with all rentals matching these parameters. Rentals include rental period, location, and basic car information.

Alternative flows:
2a. Customer has no rentals.
    1. System informs the customer that there are no rentals.

Information Requirements: None

```
