# Use Case "View Car Information"

```
Title: View Car Information

Primary Actors: Fleet manager
Secondary Actors: None

Preconditions:
    - Car exits in the system.
    
Postconditions: None

Flow:
1. Fleet manager enters VIN.
2. System presents basic car information of the car with this VIN.

Alternative flows:
1a. Fleet manager provides an invalid VIN.
    1. System prompts customer to enter antother VIN.


Information Requirements: None
```
