# Use Case "List Available Cars"

```
Title: List Available Cars

Primary Actors: Customer
Secondary Actors: None

Preconditions: None

Postconditions: None

Flow:
1. Customer enters a rental period and location to list all available cars matching the entered parameters.
2. System presents all available cars and their basic car information for the given parameters.

Alternative flows:
2a. No matches.
    1. System informs the customer that there are no available cars with the given parameters.

Information Requirements: None

```
